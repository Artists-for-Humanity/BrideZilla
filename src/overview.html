Home Overview Game

Run to each station to achieve each demand that the bride has given you in a timely fashion. Click the spacebar to get the item you need. Complete each of the mini games.

Move using the arrow keys ( up, down, left, right ). Move to the station that the bride has demanded. Collect the item with the spacebar. Press the spacebar to give the item to the bride. The timer starts when the bride gives the first demand. Complete all this before the timer runs out.
